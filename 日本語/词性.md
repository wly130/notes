### 动词

#### 分类

| I 类       | `ます形` 最后一个音节为 `い段`                             |
| ---------- | ---------------------------------------------------------- |
| **II 类**  | **`ます形` 最后一个音节大部分为 `え段` , 小部分为 `い段`** |
| **III 类** | **以 `名词 + します` 结尾**                                |

#### 变形规律

##### I 类动词

| あ段                                           | い段         | う段         | え段                           | お段         |
| ---------------------------------------------- | ------------ | ------------ | ------------------------------ | ------------ |
| **`ない形`/`被动形`/ `使役形` / `使役被动形`** | **`ます形`** | **`辞書形`** | **`假定形`/`可能形`/`命令形`** | **`意志形`** |
| **あ→わ**                                      | **い**       | **う**       | **え**                         | **お**       |
| **か**                                         | **き**       | **く**       | **け**                         | **こ**       |
| **が**                                         | **ぎ**       | **ぐ**       | **げ**                         | **ご**       |
| **さ**                                         | **し**       | **す**       | **せ**                         | **そ**       |
| **た**                                         | **ち**       | **つ**       | **て**                         | **と**       |
| **な**                                         | **に**       | **ぬ**       | **ね**                         | **の**       |
| **ば**                                         | **び**       | **ぶ**       | **べ**                         | **ぼ**       |
| **ま**                                         | **み**       | **む**       | **め**                         | **も**       |
| **ら**                                         | **り**       | **る**       | **れ**                         | **ろ**       |

| ます形                          | て形                    | た形                    |
| ------------------------------- | ----------------------- | ----------------------- |
| **最后一个音节为 `い、ち、り`** | **去掉 + `って`**       | **去掉 + `った`**       |
| **最后一个音节为 `み、び、に`** | **去掉 + `んで`**       | **去掉 + `んだ`**       |
| **最后一个音节为 `き、ぎ`**     | **去掉 + `いて、いで`** | **去掉 + `いた、いだ`** |
| **最后一个音节为 `し`**         | **+ `て`**              | **+ `た`**              |
