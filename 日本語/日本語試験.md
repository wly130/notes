## 日本語能力認定試験 (JLPT)

### 言語知識

#### 用时

-   **文字・词汇**（約13题）建议总时间 **10–12分钟**

| 题型     | 题数 | 每题时间 | 技巧       |
| -------- | ---- | -------- | ---------- |
| 汉字读音 | 5题  | 15秒     | 一眼判断   |
| 词义选择 | 5题  | 20秒     | 不会马上跳 |
| 近义词   | 3题  | 25秒     | 看句子语感 |

-   **语法**（約22题）建议总时间 **20–25分钟**

| 题型     | 题数 | 每题时间 | 特点       |
| -------- | ---- | -------- | ---------- |
| 语法选择 | 12题 | 30秒     | 基本语法   |
| 句子排序 | 5题  | 1分钟    | 星号题     |
| 文章语法 | 5题  | 1分钟    | 小短文填空 |

-   **句子排序一定要先看结尾**

### 読解

#### 用时

-   建议总时间 **60–65分钟**，平均1题 **1.5分钟**

| 题型     | 数量 | 每篇时间  | 做题方法   |
| -------- | ---- | --------- | ---------- |
| 短篇阅读 | 4篇  | 4–5分钟   | 先看题目   |
| 中篇阅读 | 2篇  | 8分钟     | 找关键词   |
| 长篇阅读 | 1篇  | 12–15分钟 | 看段落结构 |
| 信息检索 | 1题  | 4分钟     | 表格题     |

-   **遇到不会的词不要停**
-   **长句一定要切分**

#### 短篇阅读

1.   **先看题目**
2.   **找题目关键词**
3.   **回文章定位**

只读 **相关段落**，不要全文读

#### 中篇阅读

-   看 **段落结构**

-   日本文章结构通常是 **问题提出 > 说明 > 举例 > 作者结论**

-   所以,**答案80%在最后一段**

关键词:  **つまり**，**要するに**，**したがって**，**このことから**

#### 长篇阅读

-   **不要**从头逐句翻译
-   只看每段 **第一句** 和 每段 **最后一句**

#### 信息检索

1. **先看问题**
2. **找关键词**
3. **在文章里扫描**

### 聴解

#### 用时

-   固定**50分钟**

-   只播放一次
-   没时间思考

| 题型         | 特点           | 难度 |
| ------------ | -------------- | ---- |
| 課題理解     | 听对话解决问题 | ⭐⭐   |
| ポイント理解 | 听重点信息     | ⭐⭐   |
| 概要理解     | 听大意         | ⭐⭐⭐  |
| 即時応答     | 快速反应       | ⭐⭐⭐  |
| 統合理解     | 长对话         | ⭐⭐⭐⭐ |

-   **不要听每个词，要听“信息”**

需要听的只有 4 个东西

- **人物关系**（谁在说话）
- **问题是什么**
- **发生什么事情**
- **最后的决定**

### 做题节奏

| 阶段 | 时间   |
| ---- | ------ |
| 词汇 | 10分钟 |
| 语法 | 20分钟 |
| 阅读 | 65分钟 |
| 检查 | 10分钟 |

## 読解練習

### 精読 (每日30分钟-1小时)

每日 **2篇短篇阅读 + 2篇中篇阅读 + 1篇长篇阅读**

1. **先看问题**
2. **带着问题，快速阅读一遍文章，选出答案**
3. **找出不会的单词和语法，进行记忆和理解**
4. **重新阅读一遍文章**

### 泛読 (每日30分钟-1小时)

1.   **找新闻，文章，小说等资源**

2.   **先大概看一遍，理解剧情**

3.   **再完整的看一遍**

## 聴解練習

### 精听 (每日30分钟-1小时)

**第一阶段**

每日听 **30句**

1.   **只听一句话**
2.   **边听边写**
3.   **看原文**
4.   **听不懂的地方，反复的听，直到全部听懂**
5.   **再完整的听一遍句子**

**第二阶段 (第一阶段能够听懂80%以上)**

每日听 **5分钟左右的录音**

1. **听一遍对话或文章，做题**
2. **再听一遍，写关键词**
3. **看原文**
4. **听不懂的地方，反复的听，直到全部听懂，并理解其中的关系和逻辑**
5. **再完整的听一遍对话或文章**

### 泛听 (每日30分钟-1小时)

1.   **找新闻，播客，电视剧，动漫等资源**
2.   **先大概听一遍，理解剧情**
3.   **关闭字幕，再听一遍**

**注意**

-   **认真的听，不要边听边做其他事**
-   **尽量找有日语字幕的资源**
