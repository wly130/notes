### 搜索技巧

- **针对百度,Google等搜索引擎**

#### 搜索指定网站内容

```shell
文本 site:网址
```

#### 搜索指定文件类型

```shell
文本 flietype:文件类型
```

#### 在标题中搜索

```shell
intitle:标题
```

#### 在URL中搜索

```shell
inurl:关键词
```

#### 在正文中搜索

```shell
intext:关键词
```

#### 排除关键词搜索

```shell
文本 -关键词
```

#### 包含关键词搜索

```shell
文本 +关键词
```

#### 完整关键词搜索

```
"关键词"
```

#### 模糊搜索(*)

```shell
关*词
```

#### 搜索包含多个关键词

```shell
关键词1 关键词
```

#### 搜索包含某一个关键词

```shell
关键词 OR 关键词
```
