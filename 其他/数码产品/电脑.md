##### 命名规则

<img src="../../img/intelCPU命名方式.jpg" style="margin:0;width:35rem">

- **Intel Core i7-14700K**

| 命名部分    | 意义                                                    |
| ----------- | ------------------------------------------------------- |
| **Core i7** | 系列／性能等级（i3 入门 → i5 主流 → i7 高端 → i9 旗舰） |
| **14**      | 代数（14 代）                                           |
| **700**     | SKU代号（同代中性能排名，高数字一般性能更强）           |
| **K**       | 后缀字母，代表特殊功能定位                              |

| 后缀        | 意义                                            |
| ----------- | ----------------------------------------------- |
| **K**       | 解锁倍频，可超频                                |
| **F**       | 无集成显卡（需独立显卡）                        |
| **KF**      | 既可超频又无集成显卡                            |
| **U**       | 超低功耗（轻薄本常见）                          |
| **H/HK/HX** | 高性能笔记本系列（HK/HX 多为更高性能/可超频版） |
| **T**       | 低功耗/低频率桌面版                             |
| **E**       | 嵌入式平台                                      |
| **G1-G7**   | 核显性能等级                                    |

##### 优点

- **性能高, 功耗低**
- **稳定, 兼容性好**
- **单核心性能强**
- **价格高, 超频空间小**

#### AMD

##### 命名规则

<img src="../../img/AMDCPU命名方式.jpg" style="margin:0;width:35rem">

-   **AMD Ryzen 7 5800X**

| 命名部分            | 意义                                                         |
| ------------------- | ------------------------------------------------------------ |
| **Ryzen 7**         | 系列／性能等级（Ryzen 3 → Ryzen 5 → Ryzen 7 → Ryzen 9 越大性能越强） |
| **5**               | 代数（本例为 5 代，也是架构代数）                            |
| **800**             | SKU 排位（代内性能顺序）                                     |
| **X / X3D / F / G** | 后缀，代表特殊特性                                           |

| 后缀                | 意义                                                         |
| ------------------- | ------------------------------------------------------------ |
| **X**               | 主要用于桌面款，性能略高                                     |
| **X3D**             | 带 3D V-Cache 技术（大缓存，游戏性能更好）                   |
| **G**               | 搭载集成显卡（APU）                                          |
| **F**               | 桌面款但不带集成显卡                                         |
| **无后缀**          | 标准版，比如普通桌面 Ryzen CPU                               |
| **U / H / HS / HX** | 笔记本系列（**U**：低功耗；**H**：高性能；**HS/HX**：更高性能/可超频）([zhihu.com](https://www.zhihu.com/tardis/bd/art/596584769?utm_source=chatgpt.com)) |

##### 优点

- **价格低, 性价比高**
- **超频空间大**
- **多核心性能强**

#### AMD

##### 命名规则

-   **AMD Radeon RX 7800 XT**

| 组成部分                    | 意义                                          |
| --------------------------- | --------------------------------------------- |
| **Radeon**                  | 品牌/产品线                                   |
| **RX**                      | 表示 Radeon 系列独显（主要面向游戏/消费市场） |
| **7**                       | 代数/系列（例如 6000、7000、9000 系列）       |
| **800**                     | 层级定位，数字越大性能越高                    |
| **GRE / XT / XTX / 无后缀** | 性能版本（无后缀 < GRE < XT < XTX）           |

#### NVIDIA

##### 命名规则

-   **NVIDIA GeForce RTX 4090 Ti**

| 组成部分                | 意义                                               |
| ----------------------- | -------------------------------------------------- |
| **GeForce**             | 品牌/产品线（GeForce 代表游戏/消费级）             |
| **RTX**                 | 系列标识（RTX 表示支持光追 Ray Tracing & AI 特性） |
| **40**                  | 世代/系列代数（40 系列是第四代）                   |
| **90**                  | 层级/档位（数字越大性能越强）                      |
| **Ti / Super / 无后缀** | 性能版本（无后缀 < Super < Ti）                    |
