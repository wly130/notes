## Ionic

**[Ionic](https://ionicframework.com/) 是一个开源 UI 工具包，用于使用 Web 技术构建高性能、高质量的移动和桌面应用程序，并集成了Angular、React和Vue等流行框架**

### 初始化

```shell
npm install -g ionic@latest  #安装
ionic start 项目名  #创建项目
ionic serve  #运行项目
```

**模板类型**

| tabs      | 带底部导航栏 |
| -------- | ---- |
| **sidemenu** | **带侧边菜单和内容区域导航** |
| **blank** | **空白** |
| **list** | **带列表** |
