## 日语语法

| 中文       | 日语                   | 例句                           |
| ---------- | ---------------------- | ------------------------------ |
| **…是…**   | **…は…です**           | 私**は**学生**です**           |
| **…不是…** | **…は…ではありません** | 私**は**先生**ではありません** |
| **…和…**   | **…と…**               | 先生**と**生徒                 |
| **…也是…** | **…も…です**           | 彼**も**学生**です**           |
| **从…到…** | **…から…まで**         | 上海**から**北京**まで**       |
