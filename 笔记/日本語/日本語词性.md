## 日本語词性

### 动词

#### 动词分类

| I 类       | `ます形` 最后一个音节为 `い段`                             |
| ---------- | ---------------------------------------------------------- |
| **II 类**  | **`ます形` 最后一个音节大部分为 `え段` , 小部分为 `い段`** |
| **III 类** | **以 `名词 + します` 结尾**                                |

#### 动词禁止形

- **`字典形` + `な`**

#### 变形规律

##### I 类动词

| あ段                           | い段         | う段         | え段                           | お段         |
| ------------------------------ | ------------ | ------------ | ------------------------------ | ------------ |
| **`ない形`/`被动形`/`使役形`** | **`ます形`** | **`字典形`** | **`假定形`/`可能形`/`命令形`** | **`意志形`** |
| **あ→わ**                      | **い**       | **う**       | **え**                         | **お**       |
| **か**                         | **き**       | **く**       | **け**                         | **こ**       |
| **が**                         | **ぎ**       | **ぐ**       | **げ**                         | **ご**       |
| **さ**                         | **し**       | **す**       | **せ**                         | **そ**       |
| **た**                         | **ち**       | **つ**       | **て**                         | **と**       |
| **な**                         | **に**       | **ぬ**       | **ね**                         | **の**       |
| **ば**                         | **び**       | **ぶ**       | **べ**                         | **ぼ**       |
| **ま**                         | **み**       | **む**       | **め**                         | **も**       |
| **ら**                         | **り**       | **る**       | **れ**                         | **ろ**       |

| ない形     | 变形 + `ない`     |
| ---------- | ----------------- |
| **ます形** | **变形 + `ます`** |
| **字典形** | **变形**          |
| **假定形** | **变形 + `ば`**   |
| **意志形** | **变形 + `う`**   |
| **被动形** | **变形 + `れる`** |
| **可能形** | **变形 + `る`**   |
| **使役形** | **变形 + `せる`** |
| **命令形** | **变形**          |

| ます形                          | て形                    | た形                    |
| ------------------------------- | ----------------------- | ----------------------- |
| **最后一个音节为 `い、ち、り`** | **去掉 + `って`**       | **去掉 + `った`**       |
| **最后一个音节为 `み、び、に`** | **去掉 + `んで`**       | **去掉 + `んだ`**       |
| **最后一个音节为 `き、ぎ`**     | **去掉 + `いて、いで`** | **去掉 + `いた、いだ`** |
| **最后一个音节为 `し`**         | **+ `て`**              | **+ `た`**              |

##### II 类动词

| て形       | ます形 + `て`               |
| ---------- | --------------------------- |
| **た形**   | **ます形 + `た`**           |
| **ます形** | **ます形 + `ます`**         |
| **ない形** | **ます形 + `ない`**         |
| **字典形** | **ます形 + `する`**         |
| **假定形** | **字典形去`る` + `れば`**   |
| **意志形** | **字典形去`る` + `よう`**   |
| **被动形** | **字典形去`る` + `られる`** |
| **可能形** | **字典形去`る` + `られる`** |
| **使役形** | **字典形去`る`+ `させる`**  |
| **命令形** | **字典形去`る` + `ろ`**     |

##### III 类动词

| て形       | ます形 + `て`                 |
| ---------- | ----------------------------- |
| **た形**   | **ます形 + `た`**             |
| **ない形** | **ます形 + `ない`**           |
| **字典形** | **ます形 + `る`**             |
| **假定形** | **字典形去`する` + `すれば`** |
| **意志形** | **字典形去`する` + `しよう`** |
| **被动形** | **字典形去`する` + `される`** |
| **可能形** | **字典形去`する` + `できる`** |
| **使役形** | **字典形去`する` + `させる`** |
| **命令形** | **字典形去`する` + `しろ`**   |
